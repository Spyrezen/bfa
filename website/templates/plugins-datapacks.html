Plugins

ActionBarHealth

Displays Mob health when hovering crosshair on a mob

AutoReplant

Automatically replants seeds if you have them in your inventory

BetterNav

A navigation plugin which can tell you exactly where to go and how much farther you have until arrival. The base command is /bn. Use /bn help for a list of commands.

BetterTeams

Allows you and your friends to create a team for custom team chat as well as flairs on your username! Do /team create [name] to make your own team! Check the link for a list of commands.

Bottled Exp

This plugin allows you to bottle your XP for safekeeping! Use /bottle

BlockLocker

A classic must have plugin for SMPs. Prevents item theft. Allows signs to be used as personal locks on any container such as chests or furnaces as well as doors and trapdoors. You can give more players permission with the command /blocklocker (linenumber) add (player name). You can also connect signs above and below for additional players.

Brewery

A fun plugin allowing you to brew wine and beer in minecraft, no joke. Here’s a video showing the process, it’s a bit complex. Though you start by placing a cauldron filled with water over a campfire. Then insert some ingredients such as sugarcane, fermented spider eyes, etc according to the recipes on the wiki and the video.

Bstats

A data collection plugin for the server which gathers info about the playercount, online mode, minecraft version, clients, and more. All data is stored anonymously.

CoreProtect

Allows inspection of any recent blocks broken or placed. Helps identify griefers or thieves. Use /co i to enter inspector mode.

Cmilib

Just a dependency plugin for BottleXP

DiscordSRV

Connects minecraft chat to discord! A revolutionary plugin for communication and server alerts!

DurabilityAlert

Alerts you when the durability of a tool you are using is low!

EssentialsX

The basics when it comes to plugins for a minecraft server, such as /fly, /tpa, /home, /warp, /godmode, and more!

GriefProtection

A quintessential plugin for SMPs to prevent all forms of grief. Is done by players claiming land with a golden shovel. Base command is /claim. Here is a video going more in-depth

ModernMoves

Allows you to sit down in minecraft! Is just for fun and aesthetics!

Harbor

Makes it so 1-3 players have to sleep to skip the night! Otherwise everyone in the server would need to be in a bed!

Harvest XP

Allows you to gain xp after harvesting crops.

InventoryRollback

A crucial plugin for admins, creating inventory and e-chest backups for all players. If died from lag or an extenuating circumstance you are entitled for your inventory to be restored; contact an admin. Base command is /ir restore.

LuckPerms

Denotes command and plugin permissions to all players.

NBTapi

Allows custom NBT tags for itemstacks, entities, and more!

OpenInv

A vital plugin for admins, enabling them to peek into any player’s inventory or e-chest, whether online or offline. We’ve got our eyes on you… We’re always watching. ALWAYS watching…

PerPlayerDifficulty

Allows you to set your personal difficulty, lowering the damage mobs do to you. Server difficulty is hard, however.

PlanPlayerAnalytics

Gives fun server stats about playerbase and activity! Also lag info and plugins!

PlayerHeads

Gives playerheads!

PvpManager

Allows management of personal pvp! Command to toggle it off or on is /pvp. Is also the base command for other cool things like an aura and blood animation! Also detects disconnects! So don’t cheat!

SimpleRename

Allows renaming items and entities and glowing! Base command is /simplerename

TraderNotify

Notifies you when a wandering trader appears because they are annoying.

InvisItemFrames

Makes invisible item frames acquirable in survival mode via crafting! Is done with lingering potions invisibility surrounded by item frames!

UltimateTimber

The ultimate tree chopper plugin! Use an axe to chop an entire tree down at once! But beware, the tree will tumble on you! Also subtracts the right durability. Can be toggled on or off with /ultimatetimber toggle

Vault

An economy plugin.

Veinminer

A very useful mining plugin used by sneaking and then mining an ore vein. Can be toggled on or off with /veinminer toggle. Be wary that vein miner works on gravel, sand, and magma blocks so you might want to turn it off if you are anywhere near lava!

WorldEdit

The best world modification plugin. Is for admins only.

WorldGuard

A plugin used for setting complex world borders and regions in a world. Only for admin use.

 

Datapacks

 

AFKDisplay

Displays when a player is afk. Use /afk to activate.

Armor Statues

Allows custom armorstand positions.

Cauldron Concrete

A datapack that makes it so when you drop concrete powder into a cauldron, it turns it into concrete.

CoordinatesHUD

A simple datapack that shows you your coords when activated.

NetherPortalCoords

Shows you where a nether portal is. Very useful for netherportal linking.

DoubleShulkerDrops

Makes is so when you kill a shulker, it drops two shulker shells.

DragonDrops

When the dragon is killed, it drops an elytra.

MoreMobHeads

Adds obtainable mobheads.

SilenceMobs

Allows mobs to be silence so they make no noise. Rename a nametag, silence, and name a mob with it.

PlayerHeadDrops

Adds obtainable playerheads.

CraftableCoralBlocks

Allows you to craft coral blocks.

CraftableNameTags

Allows you to craft name tags

CraftableHorseArmor

Allows you to craft horse armor

CraftableGravel

Allows you to craft gravel

UnpackableIce

Allows you to turn blue ice into packed ice and packed ice into ice.

UnpackableWool

Allows you to turn a wool block into 4 string.

FastVillagerCuring

Decreases the time needed for a villager to be cured.

HowManyBees

Displays how many bees are in a bee hive.

GlowingGlowSquids

Makes glow squid glow.

VillagerLeads

Allows you to attach leads to villagers for easier transportation

BetterSlabBreaking

Makes it so if you break the bottom part of two slabs, it breaks the bottom slab and vice versa.

BetterLightning

It adds fun new effects when lighting strikes different entities. For example, “If lightning strikes a tree, the logs become stripped logs, similar to how in real life lightning can blow the bark off of tree trunks.” There is a more comprehensive list of all the effects that lightning now has in the link above.

ShipsOutOnTheOceans

Spawns new ships in the ocean that can either be friendly or hostile!

Gateways

Allows you to travel to places you’ve already visited by placing one gateway at the start and end location and linking them together with a gateway linker. View the link for the crafting recipes.

EndAboveOverworld

Changes the void in the End so that if you fall off, you will fall into those exact coordinates in the overworld! No more falling into an endless void and losing all your items.

LimitlessTrades

Makes it so villagers will never have to restock and can trade with you infinitely. This makes villager trading much faster, however you still need the same amount of materials to trade as you would normally. Note: There will be an X and the GUI will say that you cannot trade anymore however you will still be able to trade!

LavaSponge

A sponge but for lava. Scoria
